# 원문 07 — [참고] AI Agent 대상영역 분석 Template

> 업무 흐름을 Swimlane(가로 5열) 형태로 그리고, 각 Task 박스를 **AI Agent 수행 / Human 수행**으로 색·아이콘 구분하는 빈 템플릿.

## 구조
- 최상단: 5개의 단계(Phase) 헤더 (회색 박스, 좌→우 화살표로 연결)
- 각 컬럼 아래로 Task 박스가 수직 배열되고, 박스 간 연결선으로 흐름 표현
- 박스는 두 종류 아이콘으로 구분:
  - 🤖 **AI Agent 수행 업무**
  - 🧑 **Human 수행 업무**

## 범례 (Legend)
| 아이콘 | 의미 |
|--------|------|
| 🤖 (로봇) | AI Agent 수행 업무 |
| 🧑 (사람) | Human 수행 업무 |

**해석 포인트**: To-Be 프로세스를 그릴 때 "AI가 하는 칸"과 "사람이 하는 칸"을 한 장에 시각화 → 원문 10의 Check 항목 "AI·사람 역할의 명확한 구분"을 실제로 작성하는 도구.
