LLM-only 답안 캘리브레이션 샘플. 예상 31~44점. LLM 카피의 실패 패턴을 모두 노출.

읽는 순서 (역할별)

출제자/주최 측 → Spec → 모범 답안 → LLM-only 답안 → 문제지 (전체 그림 → 채점 기준 합의 → 참가자 안내 확정)

채점자 → Spec § 6 채점 rubric → 모범 답안 & LLM-only 답안 (점수 분포 합의) → 실제 답안 채점

참가자 → 문제지(exam-paper)만 보면 됨. 다른 문서는 시험 종료 후 공개.
