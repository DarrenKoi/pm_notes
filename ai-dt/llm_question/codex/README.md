# LLM Question Codex Notes

이 폴더는 `bgk.txt`를 기준으로 새로 시작한 45분 LLM 리포트 경진대회 출제 논의를 관리한다.

기존 `ai-dt/llm_question/contest-spec.md`, `exam-paper.md`, 샘플 답안들은 이전 90분안 산출물로 유지하고, 새 논의는 이 폴더에서 별도로 발전시킨다.

## Files

- [index.html](./index.html): 사람이 읽기 편한 HTML 요약 페이지.
- [contest-codex.css](./contest-codex.css): HTML 요약 페이지 스타일.
- [CONTEXT_CODEX.md](./CONTEXT_CODEX.md): 이번 경진대회 논의에서 확정된 용어와 관계.
- [contest-design-notes.md](./contest-design-notes.md): 현재까지 확정된 45분안 출제 설계.
- [open-questions.md](./open-questions.md): 다음 grilling에서 이어서 결정할 질문.

## Current Direction

- 전체 운영 시간은 50분이다.
- 실제 작성 시간은 45분이고, 마지막 5분은 제출 시간이다.
- 대회의 핵심은 LLM 없이 혼자 쓰는 도메인 지식 평가가 아니라, LLM을 활용해 제출 가능한 좋은 리포트를 완성하는 능력이다.
- 참가자는 본인 전체 직무 중 반복적으로 수행하는 업무 1개를 선택한다.
- 답안은 3개 파트로 압축한다.
- 참가자는 최종 리포트에 중요하게 작용한 핵심 프롬프트 2~3개와 그 활용 흔적을 함께 제출한다.
