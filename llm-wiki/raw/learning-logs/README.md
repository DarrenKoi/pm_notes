- 파일명: `YYYYMMDD-<topic>.md` (예: `20260501-langgraph-state.md`)
- 길이: 3~5 줄도 OK. 완성도보다 빈도가 중요.
- 막혔던 지점, 헷갈린 개념, 검색한 키워드, 참고 링크 등.
- 템플릿: [`../../_templates/learning-log.md`](../../_templates/learning-log.md)

ingest 시 LLM 이 `wiki/concepts/<topic>.md` 로 정제한다.
